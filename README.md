# Beverage Tracking

## Requirements

Design and implement an HTTP API that controls an imaginary internet-connected coffee machine


## Technologies

- ASP.NET Core 3.1
- ASP.NET WebApi Core
- ASP.NET Core Middleware
- Autofac
- Polly Cache
- Polly Retry
- Serilog
- xUnit
- Moq

## How to run
```
dotnet user-secrets set "OpenWeatherOptions:ApiId" "390aff1a1e3f9edfd40199bc987fad58"
```
